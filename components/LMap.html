# LMap

基本组件，包含并包装所有其他组件。

提示：如果<l-map>不在根节点上会报错

# Props

Prop name Description Type Values Default
options object - {}
center 地图的中心，支持 .sync 修饰符 object|array - () => [0, 0]
bounds 地图的边界，支持 .sync 修改器 array|object - null
maxBounds 地图的最大边界 array|object - null
zoom 地图的缩放，支持 .sync 修改器 number - 0
minZoom 地图的最小缩放 number - null
maxZoom 地图的最大缩放 number - null
paddingBottomRight 地图的填充底部右侧 array - null

# Events

Event name Type Description
ready object 当地图 ready 时触发
update:zoom number,string 当地图 center 更新时(updated)时触发
update:center object,array 当地图 center 更新时(updated)时触发
update:bounds object 当地图 bounds 更新时(updated)时触发
